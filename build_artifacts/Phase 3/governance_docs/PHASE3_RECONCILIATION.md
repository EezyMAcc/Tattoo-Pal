**Validated pair behaviour** (confirmed by instantiation):

- `auth=AuthSettings(...)` without `token_verifier` → `ValueError: Must specify
  either auth_server_provider or token_verifier when auth is enabled`
- `token_verifier=<verifier>` without `auth` → `ValueError: Cannot specify

- `auth=AuthSettings(...), token_verifier=<verifier>` together → constructs
  successfully; `server.settings.auth` is the `AuthSettings` instance;
  `server._auth_server_provider` is `None` (verifier path, not provider path).

**Conclusion:** `PLAN.md §3`'s construction pattern is correct:

```python
FastMCP(
    "tattoo-feed",
    instructions=...,
    auth=AuthSettings(
        issuer_url=AnyHttpUrl(auth_cfg.issuer),
        resource_server_url=AnyHttpUrl(auth_cfg.audience),
        required_scopes=auth_cfg.required_scopes or None,
    ),
    token_verifier=IdpTokenVerifier(...),
)
```

### 2b. Tool registration — `server.tool()(fn)` equivalence

- `server.tool()(my_fn)` **returns the original function unchanged** (`result is
  my_fn` is `True`) and registers it in `_tool_manager` under `my_fn.__name__`.
- This is behaviourally equivalent to using `@server.tool()` as a decorator.
- Keyword options work: `server.tool(structured_output=False, meta={...})(fn)`
  returns `fn` unchanged and registers it with those options.

**Conclusion:** `build_server` in Chunk 1 may register all 11 tools via
`server.tool()(fn)` calls (and `server.tool(structured_output=False, meta=...)(fn)`
for `next_inspiration` if needed) with no change to the callable surface.

### 2c. Resource registration — `server.resource(uri)(fn)` equivalence

- `server.resource(uri)(my_fn)` **returns the original function unchanged**
  (`result is my_fn` is `True`) and registers it in `_resource_manager` under
  `uri`.
- `_resource_manager.list_resources()` is a **synchronous** method (not async).
  The existing `test_server_widget.py` calls it without `await` — this is correct.

**Conclusion:** `build_server` may register the widget via
`server.resource(_WIDGET_URI, mime_type="text/html;profile=mcp-app")(_widget_inspiration)`.
